# Highwayman Button

A tiny standalone home-screen button for Samsung/Android.

- Opens Spotify to *Highwayman* by The Highwaymen
- Not connected to Gjøremål
- No auto-open while setting it up in Chrome

## Install on Galaxy S24

1. Open the GitHub Pages URL in Chrome.
2. Tap `⋮` → **Add to Home screen**.
3. Name it `Highwayman`.
4. Tap the home-screen icon, then press **Play album**.

Spotify album URL: https://open.spotify.com/album/6rFM5jHaGzwut3hGbG9pGU
